# 🏢 ROSE - Plataforma de Búsqueda, Canjeo y Donación de Medicamentos Oncológicos

## ✨ Descripción del Proyecto
ROSE es una aplicación diseñada para facilitar el **canjeo, donación y visualización de la disponibilidad de medicamentos oncológicos** específicos para el tratamiento del **cáncer de mama** en Venezuela. 

El objetivo principal es **conectar a pacientes o familiares** con personas y establecimientos que poseen estos medicamentos, reduciendo la dificultad de acceso y el estrés generado por la búsqueda de tratamientos esenciales.

A través de ROSE, los usuarios pueden:
- 💼 **Canjear medicamentos** entre usuarios.
- ❤️ **Donar medicamentos** a quienes los necesiten.
- 📊 **Consultar precios en el mercado**.
- 📍 **Ver ubicaciones en Caracas** donde están disponibles medicamentos de difícil acceso.

---

## 🛠️ Tecnologías Utilizadas

### **Frontend**
- **HTML** ✨ - Estructura de la aplicación.
- **Tailwind CSS** 🔄 - Framework para un diseño rápido y moderno.
- **JavaScript** 🛠 - Lógica del frontend.
- **Regex** 🔖 - Validación de datos (por ejemplo, verificación de nombres de medicamentos y códigos).
- **API Fetch** 🔄 - Conexión con el backend para obtener y enviar datos.

### **Backend**
- **Node.js** 🤖 - Entorno de ejecución del servidor.
- **MongoDB** 💾 - Base de datos para almacenar información sobre medicamentos, usuarios y ubicaciones.
- **Render** 🛠️ - Plataforma de despliegue del backend.

---

## 🔄 Flujo de Funcionamiento
1. 👤 **Registro/Login**: Los usuarios pueden registrarse y acceder a la plataforma.
2. 🛥️ **Publicación de Medicamentos**: Los usuarios pueden subir información sobre medicamentos disponibles para donación o canje.
3. 📊 **Consulta de Precios**: Los usuarios pueden ver los precios del mercado de los medicamentos.
4. 📍 **Ubicaciones**: Se muestran los lugares en **Caracas** donde están disponibles.
5. ✉️ **Contacto Directo**: Se facilita la comunicación entre donantes y beneficiarios.

## 🛠️ Instalación y Configuración

### 1. **Clonar el Repositorio**

  git clone https://github.com/gloriaalbino/ROSE
  cd ROSE

### 2. **Instalar Dependencias**
Ejecuta el siguiente comando en la carpeta del proyecto:

  npm install

### 3. **Ejecutar la Aplicación**

  npm run start

La aplicación estará disponible en: https://rose-pf.onrender.com
